# Not My Handwriting AI

## Overview
Not My Handwriting is an AI authorship verification system designed to identify whether a piece of text was written by a specific author. Utilizing advanced machine learning techniques, it analyzes writing styles and patterns to deliver accurate authorship detection.

## Tech Stack
- **Programming Language**: Python
- **Frameworks**: TensorFlow, Flask
- **Libraries**: NLTK, SpaCy, Scikit-Learn
- **Database**: SQLite

## How to Run
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Slarter03/Not-My-Handwriting.git
   cd Not-My-Handwriting
   ```
2. **Install the necessary packages**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the application**:
   ```bash
   python app.py
   ```
4. **Access the service** at `http://localhost:5000`.
